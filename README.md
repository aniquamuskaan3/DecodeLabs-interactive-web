# 🚀 Interactive Dashboard

A modern and interactive dashboard application built using HTML, CSS, and JavaScript. The project combines multiple useful tools into a single responsive interface, including task management, note-taking, a calculator, a live clock, and a motivational quote generator.

## ✨ Features

* 📊 Dashboard Overview
* ⏰ Live Digital Clock
* 💡 Random Quote Generator
* ✅ Task Manager
* 📒 Notes Section
* 🧮 Calculator
* 🌙 Dark Mode Toggle
* 💾 Local Storage Support
* 📱 Responsive Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🚀 How to Use

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Start managing tasks, saving notes, and using the dashboard tools.

## 📋 Modules Included

### Dashboard

Displays statistics, a live clock, and motivational quotes.

### Task Manager

Add and remove tasks with automatic local storage saving.

### Notes

Write and save notes directly in the browser.

### Calculator

Perform basic mathematical calculations instantly.

### Theme Switcher

Toggle between light and dark modes.
